6.本地搜索功能

需要结合插件 hexo-generator-search 来使用：

配置 _config.yml

search:
path: search.json
field: post
content: true

当你使用 $ hexo g 之后，publish 目录下就生成了search.json了。我们就可以根据这个json来做本地搜索。
